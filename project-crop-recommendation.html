Crop Recommendation Engine

Machine Learning Project

Project Overview

  • Developed an intelligent machine learning–based smart farming system that analyzes key soil and environmental parameters—such as soil nutrients (N, P, K), pH level, temperature, humidity, and rainfall—to recommend the most suitable crops for cultivation.
  • The system leverages data preprocessing, feature selection, and supervised learning algorithms to identify patterns between environmental conditions and crop productivity. By providing accurate, data-driven crop recommendations, the model helps farmers optimize land usage, reduce crop failure risk, and improve overall agricultural yield. The solution is designed to support sustainable farming practices by enabling informed decision-making, minimizing resource wastage, and adapting crop choices to changing climatic conditions.

Key Contributions

  • ML Model Development: Developed a machine learning model that recommends crops based on NPK values, pH, temperature, humidity, and rainfall.
  • High Accuracy: Used Random Forest and Decision Tree classifiers via Scikit-learn with over 92% accuracy.
  • Data Analysis: Performed data preprocessing, visualization (Seaborn), and evaluation to optimize for real-world agriculture.
  • Smart Farming: Designed for smart farming with practical applicability and improved yield prediction.

Technologies Used

Languages: Python
Libraries: Scikit-learn, Pandas, NumPy, Seaborn
Algorithms: Random Forest, Decision Tree
